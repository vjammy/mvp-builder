# Product Spec

MVP Builder is the markdown-first planning, gating, scoring, and handoff layer before AI coding starts.

## V1 scope

- Single-page local web app.
- Six profiles: beginner/intermediate/advanced × business/technical.
- Guided questionnaire.
- Markdown artifact generation.
- Ten phases with entry and exit gates.
- Readiness scorecard.
- Downloadable zip export.
- CLI generator.
